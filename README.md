## Development

```bash
# Install dependencies
yarn

# Start development server
yarn dev

# Build for production
yarn build

# Run tests
yarn test        # Unit tests
yarn test:e2e    # End-to-end tests
yarn test:e2e:ui # E2E tests with UI mode
```
